Shows service time expiration, resource, and transaction limits defined for the current account.

Service Expiration: When the limit is reached, no more input events will be accepted for the account.

Devices: When the limit is reached, no more devices can be added to the account.

Landmarks: When the limit is reached, no more landmarks can be added to the account.

Users: When the limit is reached, no more users can be added to the account.

Notification Subscribers: When the limit is reached, no more notification subscribers can be added to the account.

Events/Month: When the limit is reached, no more input events will be accepted for the current month.

Notifications/Month: When the limit is reached, no more notifications will be triggered for the current month.

Notifications Emails/Month: When the limit is reached, no more notifications emails will be sent for the current month.

Notifications Texts/Month: When the limit is reached, no more notifications texts will be sent for the current month.

Service limits are set using the SETUP - All Accounts selection.
