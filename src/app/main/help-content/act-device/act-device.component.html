Interface for viewing, updating, and creating devices.

Name: A device name meaningful to users of the account.

Device ID: A globally unique id that is included in all events sent from the device.

Map Marker Icon: Icon shapes, sizes, and colors may be customized for each device. A default icon may also be created for the account. If no icons specific to the device or if no default account icon is created, a system default icon is used.

Shape: Choice of available icon shapes.

Scale: Icon size.

Fill Color: CSS Icon color code.

Fill Opacity: Icon body transparency.

Fill Color: CSS Icon outline color code.

Outline Opacity: Icon outline transparency.

Outline Weight: Icon outline thickness.

Use Icon Default Button: When enabled, a click on this button erases the specific icon settings for the device, so that the default account icon will be used.

Make this Icon Default Button: When enabled, a click on this button creates a new default icon for the current account.
