For viewing and modifying the service constraints for the account.

Devices: The maximum number of devices permitted on the account.

Users: The maximum number of users permitted on the account.

Landmarks: The maximum number of landmarks permitted on the account.

Notification Subscribers: The maximum number of notification subscribers permitted on the account.

Events/Month: The maximum number of input events accepted per month.

Notifications Emails/Month: The maximum number of emails that can be sent per month.

Notifications Texts/Month: The maximum number of text messages that can be sent per month.
